Welcome to SIGMA2
===================================

**SIGMA2** is a tool for analysing SEM/(S)TEM data with segmentation.

SIGMA2 runs in **jupyter lab** - but no previous experience with python or jupyter is necessary to install or use SIGMA.


Check out the :doc:`getting_started` section for further information, including how to install SIGMA2 at :ref:`installation`.

One you have installed SIGMA2 - check out the :doc:`tutorials` for tutorial notebooks that demonstrate how to use SIGMA2 at :ref:`tutorials`.

.. note::

   This project is under active development.
